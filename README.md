# How to run the project

## Requirements
- Install rocket league
- Install RLBotGUI (https://rlbot.org/)
- Install python packages:
  - VirxERLU-CLib
  - tmcp
  - numpy

## Running the project
- Open RLBotGUI
- Select teamBot
- Select an opponent (Psyonix Pro)
- Press start and Rocket League should launch
